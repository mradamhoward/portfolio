Integrated Development Environments

What are IDE's? IDE's are a piece of software that assist a software developer when making changes to source code. The have assisted coding features that help to "autocomplete" class instances and variable names. They are text editors on steroids. They have integrated compilers that require only the push of a button to build your project and get it running. They also have integration support for frameworks that help setting up a project of a certain kind.
