But it was a start. It got me into the workflow and environment which I've grown to love, embrace and nurture.

When I was in college we were given the challenging task of building a media player. My media player had a slider as a progressbar, a play button, pause button, listview to show the songs. There was also a skip song button which the user could use to cycle the songs. It supported MP3 files. We were only tasked with making a media player that could load images and songs, but I went a step further. I integrated a video player into mine, where it could play .mp4 and other basic video file formats. The application incorporated a distributed system with Remote Method Invocation service available with Java. JavaFX was the application UI layer framework and allowed the binding of methods in the controller class to events in the UI classes.
